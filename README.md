# Student score predictor
## Overview
This project predicts student exam scores based on behavioral and academic factors using machine learning. The goal is to understand how different factors influence academic performance and build a predictive model 

## Description of data set
Dataset sourced from Kaggle. The dataset contain an overview of various factors affecting student performance in exams. 
Factors:
* Study habits
* Attendance
* Parental involvement
* Access to resources
* Extracurricular activities etc.
